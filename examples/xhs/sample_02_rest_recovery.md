## 完整原文

**比睡觉更能回血!(内附休息指南)**

周末滋溜一下就结束了，每到周一都想要仰天长啸:还没休息够怎么又要上班了!

大家有没有发现，平常我们做的「回血」方式都越来越没有效果了，甚至越休息越累:

周末哪怕连躺两天，还是昏昏沉沉，回到工作中没有满血复活的感觉;下班后狠狠报复性玩耍、追剧，却还是觉得空虚和疲惫;明明什么重活儿也没干，却总觉得累，对什么都提不起劲;

直到看到脑科学的研究才意识到，真正有效、能恢复精力的休息不是「停止」，而是靠大脑不同区域的「切换」。

## 风格要点抽取

### 结构骨架（场景共鸣型）

1. **标题**：反常识 + 暗示干货
   - `比 X 更 Y` 制造反差
   - `(内附休息指南)` 暗示有具体方法

2. **痛点共鸣开场**（1-2 行）
   - 真实场景：周末→周一
   - 拟声词增加生动感（"滋溜"、"仰天长啸"）

3. **代入感引导**（1 行）
   - `大家有没有发现`、`是不是这样` 等召唤词

4. **三个具体场景**（3 行排比）
   - 用分号串联，每个都是高频共鸣场景
   - 句式相似但内容递进

5. **研究反转**（1 段）
   - `直到看到 [研究/案例] 才意识到` 引出
   - 「X」不是 ❌，而是「Y」 ✅ 的对比

6. **Hashtags + @相关账号**

### emoji 使用

⚠️ **这篇没有 emoji**——这是对标账号的另一种风格。
- 适合：心理 / 状态 / 偏严肃议题
- 不适合：消费 / 选择 / 多分点信息（用示例 01 的 emoji 密度）

→ **判断规则**：议题越严肃，emoji 越少；议题越生活化，emoji 越密。

### 互动方式

这篇用「@心理薯」标签让账号被推荐到对应话题流。社创类可以参考：
- @公益薯（如果存在）
- @社会观察 类话题

### 论证链模板（可复用）

```
[周X / 工作/生活 痛点开场]
  ↓
大家有没有发现，[现象] 越来越 [问题]:
  ↓
[场景1]; [场景2]; [场景3]; ← 三个排比
  ↓
直到看到 [研究/案例] 才意识到，
真正 [反常识结论]，
不是「[误区]」，而是「[新认知]」。
  ↓
[展开 1-3 个分点，可省略，留悬念引导评论也可]
```

## 两个示例对比

| 特征 | 示例 01（可持续产品） | 示例 02（比睡觉更回血） |
|---|---|---|
| 议题类型 | 消费 / 价值观 | 心理 / 状态 |
| 长度 | ~280 字正文 | ~150 字 |
| emoji 数量 | 多（每 2 行 1 个） | 0 |
| 结构 | 对比型 | 共鸣型 |
| 核心钩子 | 场景纠结 + 数据 | 周一痛点 + 反常识 |
| 延伸阅读 | 有 | 无 |
| 互动方式 | 提问 + `💬下方交换` | 仅 @话题账号 |

- 涉及"消费 / 选择 / 价值"类原文 → 用示例 01 模板
- 涉及"心理 / 状态 / 反常识洞察"类原文 → 用示例 02 模板
