# /humanizer-zh 审校

## 输入解析

从参数中确定目标文件：

- 如果参数是 slug（如 `what-are-schools-for-now`），在 `data/output/` 下找对应的 `wechat.md`
- 如果参数是完整路径，直接使用
- 如果没有参数，询问用户要审校哪篇

## 审校流程

1. **读取文件**：读取目标 wechat.md 的完整内容
2. **扫描 AI 痕迹**：逐段检查以下问题
3. **直接修改**：将修改写回文件（不是另存新文件）
4. **输出报告**：列出改动摘要

## 检查项（SSIR 公众号专用）

### 必须修复

- [ ] **破折号（——）**：全文替换为逗号、句号或括号
- [ ] **否定排比**："不是……而是……""并非……而是……" → 改正面陈述
- [ ] **三段式排列**：同一段出现 3 次及以上的短语并列 → 改为两项或叙述句
- [ ] **AI 高频词**：`此外` `至关重要` `彰显` `格局`（抽象用法）`标志着` `不断演变` `深入探讨` `充满活力` `开创性` `见证了` → 删除或替换
- [ ] **模糊归因**："研究显示""专家认为""有观点认为" → 改具体来源或删除

### 建议修复

- [ ] **连续三句等长**：打断其中一句，拉开节奏
- [ ] **总结句收尾**：段落最后一句是"漂亮金句"→ 改成具体事实
- [ ] **翻译腔套语**："对于……而言""正是……才……""在……背景下""之所以……是因为……" → 重写
- [ ] **连续抽象名词堆叠**："X 的 Y 的 Z 机制" → 改动词表达

## 输出格式

```
✅ humanizer-zh 审校完成：{文件路径}

改动摘要（{N} 处）：
- 破折号：X 处
- AI 高频词：X 处（列出具体词）
- 否定排比：X 处
- 翻译腔：X 处
- 其他：X 处

未改动原因（若有保留）：
- [说明哪些问题未动及理由]
```
