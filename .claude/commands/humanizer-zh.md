# /humanizer-zh 审校

基于 [humanizer-zh](https://github.com/op7418/humanizer-zh)（汉化自 blader/humanizer）的通用规则，叠加 SSIR 公众号编译专项检查。

## 你的任务

1. **识别 AI 模式** — 扫描下列所有规则
2. **重写问题片段** — 用自然中文替换
3. **保留含义** — 核心论点、数据、引语一字不动
4. **直接修改文件** — 写回原文件，不另存
5. **输出报告** — 列出改动摘要与质量评分

---

## 输入解析

- 参数是 slug（如 `what-are-schools-for-now`）→ 在 `data/output/` 下找对应的 wechat.md（取最新周次）
- 参数是完整路径 → 直接使用
- 无参数 → 询问用户

## 第一层：通用去AI感规则（来自 humanizer-zh 全局 skill）

### 1. 过度强调意义与遗产

**标志词**：`标志着` `见证了` `体现了` `彰显了` `至关重要` `深深植根于` `不可磨灭` `关键转折点` `不断演变的格局` `为……奠定基础` `开创性的`

**原则**：用具体事实替代宏大宣告。不要说"标志着重要时刻"，说发生了什么。

### 2. 模糊归因

**标志词**：`专家认为` `研究显示` `观察者指出` `一些批评者认为` `行业报告显示`

**原则**：改成具体来源（人名 + 机构 + 年份），或删除。

### 3. 宣传广告式语言

**标志词**：`充满活力的` `深刻的` `迷人的` `开创性的` `令人叹为观止` `致力于` `展示`（作为夸赞动词）

**原则**：用具体描述替代，说做了什么，不说它"令人叹为观止"。

### 4. AI 高频词

`此外` `至关重要` `深入探讨` `强调`（动词滥用）`持久的` `增强` `培养`（抽象用法）`彰显` `格局`（抽象名词）`关键性的` `充满活力的` `标志着` `见证了` `不断演变的` `为……奠定基础` `开创性的` `宝贵的` `复杂性`（滥用）

**原则**：遇到就考虑删除或换成具体词。

### 5. 否定式排比

**标志**："不是……而是……""不仅仅是……更是……""并非……而是……"

**原则**：改成正面直接陈述。"沉重的节拍增加了攻击性的基调"胜过"这不只是一首歌，而是一种声明"。

### 6. 三段式法则滥用

**标志**：同一段里连续出现 3 个并列短语/句子，结构完全对称。

**原则**：两项优于三项。砍掉最弱的一项，或合并为叙述句。

### 7. 破折号滥用

**标志**：`——` 用于插入说明、转折、引出例子。

**原则**：全部改为逗号、句号、冒号或括号，视语境选择。

### 8. 填充短语

| 原文 | 改写 |
|------|------|
| 为了实现这一目标 | 为此 / 删除 |
| 值得注意的是 | 删除，直接说 |
| 在这个时间点 | 现在 |
| 具有……的能力 | 能……／可以…… |
| 对于……而言 | 删除，调整语序 |

### 9. 通用积极结论

**标志**：段落或全文以"未来可期""前景光明""令人振奋"等收尾。

**原则**：改成具体的下一步事实，或直接截断。

## 第二层：SSIR 公众号专项检查

### 翻译腔套语（必须修复）

- `对于……而言` → 删除，重组语序
- `正是……才……` → 改直接陈述
- `在……背景下` → 删除或缩写为前置短语
- `之所以……是因为……` → 调换为"因为……所以……"或合并
- `的是……而不是……` → 改正面表述

### 连续抽象名词堆叠（必须修复）

"X 的 Y 的 Z 机制/框架/路径" → 拆成动词短语，说清楚谁在做什么。

### 个性与节奏（建议修复）

- 连续三句句长相同 → 打断一句
- 段落末尾是"漂亮金句" → 改成具体事实
- 全文过于"中立报道"腔 → 保留作者原有的观点锋芒

---

## 核心原则（贯穿所有规则）

1. **删除填充短语** — 去除开场白和强调性拐杖词
2. **打破公式结构** — 避免二元对比、戏剧性分段、修辞性设置
3. **变化节奏** — 混合句子长度，两项优于三项，段落结尾多样化
4. **信任读者** — 直接陈述事实，跳过软化和手把手引导
5. **删除金句** — 听起来像可引用的语句，重写它


### 质量评分（审校后）

| 维度 | 评估标准 | 得分 |
|------|----------|------|
| **直接性** | 直接陈述事实还是绕圈宣告？ | /10 |
| **节奏** | 句子长度是否变化？ | /10 |
| **信任度** | 是否尊重读者智慧？ | /10 |
| **真实性** | 听起来像真人说话吗？ | /10 |
| **精炼度** | 还有可删减的内容吗？ | /10 |
| **总分** | | **/50** |

- 45–50：优秀，AI 痕迹已清除
- 35–44：良好，仍有改进空间
- 低于 35：需要重新修订
